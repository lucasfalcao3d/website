Black Cat

Category: Personal Work
Software: Blender and Substance Painter


Not sure where the idea exactly come from, but my goal on this project was create a cat with nice looking fur and cats loves windows and sun, so here is the result.


Everything was done using Blender, Cycles and the Auto Eye. The tree used in this work was made by the creator winbrookhouse, so big thanks to him.
